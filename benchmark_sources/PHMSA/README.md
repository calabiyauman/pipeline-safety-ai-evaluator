# PHMSA (Pipeline and Hazardous Materials Safety Administration) Benchmark Test Sources

## Overview
PHMSA is the federal agency responsible for regulating pipeline safety in the United States. Their regulations in 49 CFR Parts 190-199 govern all aspects of pipeline operations, with 49 CFR 192 being the primary regulation for natural gas pipelines.

## Sample Test Questions (To Be Developed)

### Question 1: MAOP Determination
**Scenario**: A natural gas transmission pipeline was constructed in 1970 with the following records:
- Original test pressure: 1,500 psig
- Pipe specification: API 5L Grade X52, 0.375-inch wall, 20-inch OD
- Class location at time of construction: Class 1
- Current class location: Class 3 (due to development)

**Task**: Determine the Maximum Allowable Operating Pressure (MAOP) per 49 CFR 192.

**Expected Elements**:
- 49 CFR 192.619 calculation
- Design pressure vs. test pressure
- Class location change implications
- Potential need for pressure reduction

**Standards**: 49 CFR 192.619, 192.611

---

### Question 2: High Consequence Area Identification
**Scenario**: A pipeline operator is conducting an HCA identification study. The pipeline segment has:
- Potential Impact Radius (PIR): 800 feet
- Identified sites: School (1,200 ft), Hospital (600 ft), Residential area (500 ft with 75 buildings)

**Task**: Determine which sites are within the HCA and identify the HCA boundaries.

**Expected Elements**:
- PIR calculation methodology
- HCA identification criteria (192.903)
- Class location method vs. PIR method
- Documentation requirements

**Standards**: 49 CFR 192.903, 192.905

---

### Question 3: Leak Classification and Response
**Scenario**: A leak investigation identifies the following conditions at various locations:
- Location A: 40% LEL in confined space (meter set)
- Location B: 80% LEL in unconfined area
- Location C: 100% LEL with active leak (grade 3)

**Task**: Classify each leak per 49 CFR 192 and determine required response times.

**Expected Elements**:
- Grade 1, 2, 3 leak definitions
- Response time requirements
- Repair criteria
- Documentation requirements

**Standards**: 49 CFR 192.703, 192.705, 192.707

---

### Question 4: Operator Qualification Requirements
**Scenario**: A contractor will be performing the following tasks on a gas pipeline:
- Welding on pipeline main
- Operating valves for isolation
- Installing hot tap
- Conducting leak surveys

**Task**: Determine which tasks are "covered tasks" requiring operator qualification and describe the evaluation requirements.

**Expected Elements**:
- Covered task identification (Appendix A to Subpart N)
- Performance vs. knowledge evaluation
- Abnormal operating conditions (AOCs)
- Recordkeeping requirements

**Standards**: 49 CFR 192.805, 192.807, 192.809

---

### Question 5: Integrity Management Assessment Methods
**Scenario**: A gas transmission pipeline in an HCA requires integrity assessment. The pipeline:
- Has never had an inline inspection
- Has some coating damage observed during surveys
- Has cathodic protection readings showing some areas below -850 mV

**Task**: Determine appropriate assessment methods per 49 CFR 192, Subpart O.

**Expected Elements**:
- Baseline assessment requirements
- Assessment method selection (ILI, pressure test, direct assessment)
- External corrosion direct assessment (ECDA) procedures
- Follow-up assessment intervals

**Standards**: 49 CFR 192.917, 192.925, 192.937

## Notes

- 49 CFR 192 is the minimum federal safety standard
- States can adopt additional requirements
- PHMSA conducts inspections and can issue enforcement actions
- Incident reporting required within specific timeframes (Part 191)
- Integrity Management requirements apply to HCAs (Subpart O)
- Operator Qualification is required for all covered tasks (Subpart N)
- PHMSA frequently issues advisory bulletins on safety topics
