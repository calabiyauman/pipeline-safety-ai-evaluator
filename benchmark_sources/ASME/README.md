# ASME (American Society of Mechanical Engineers) Benchmark Test Sources

## Overview
ASME codes are the foundation of pipeline design, construction, and inspection in North America. B31.8 is the primary code for gas transmission and distribution pipelines.

## Sample Test Questions (To Be Developed)

### Question 1: Pressure Design
**Scenario**: A natural gas transmission pipeline is being designed with the following parameters:
- Material: API 5L Grade X70
- Nominal wall thickness: 0.500 inches
- Outside diameter: 36 inches
- Maximum operating pressure: 1,440 psig
- Design factor (F): 0.60 (Class 2 location)
- Longitudinal joint factor (E): 1.0 (seamless)
- Temperature derating factor (T): 1.0

**Task**: Verify if the pipe wall thickness is adequate per ASME B31.8.

**Expected Elements**:
- Barlow's formula application
- SMYS (Specified Minimum Yield Strength) for X70 = 70,000 psi
- Calculation of required wall thickness
- Comparison with actual thickness
- Safety margin assessment

**Formula**: t = (P × D) / (2 × F × E × S × T)

**Standards**: ASME B31.8, Chapter I, Section 841.1

---

### Question 2: Class Location Determination
**Scenario**: A pipeline route passes through several areas:
- Mile 0-5: 12 buildings intended for human occupancy within 220 yards
- Mile 5-10: 3 buildings within 220 yards
- Mile 10-15: 35 buildings within 220 yards (suburban housing development)
- Mile 15-20: Undeveloped rural area

**Task**: Determine the class location for each section and the corresponding design factor.

**Expected Elements**:
- Class 1, 2, 3, 4 definitions per B31.8
- Building count methodology
- Design factor assignment (F = 0.72, 0.60, 0.50, 0.40)
- Documentation requirements

**Standards**: ASME B31.8, Chapter I, Section 840.2

---

### Question 3: Hydrostatic Testing Requirements
**Scenario**: A new 20-inch natural gas transmission pipeline has been constructed:
- Material: API 5L Grade X52
- Wall thickness: 0.375 inches
- Class location: Class 3
- Elevation change: +600 feet from start to end

**Task**: Determine the hydrostatic test pressure per ASME B31.8.

**Expected Elements**:
- Test pressure calculation (1.25 × MAOP for Class 3)
- Elevation correction (0.433 psi/ft for water)
- Hold time requirements
- Acceptance criteria

**Standards**: ASME B31.8, Chapter VI, Section 841.3

---

### Question 4: Welding Requirements
**Scenario**: A pipeline project requires field welding of API 5L Grade X65 pipe with 0.250-inch wall thickness. The welding will be performed in winter conditions with ambient temperature of 20°F.

**Task**: Determine the welding requirements per ASME B31.8 including:
- Preheat requirements
- Welding procedure requirements
- Inspector qualifications
- Non-destructive examination requirements

**Expected Elements**:
- Preheat temperature based on material grade and thickness
- WPS/PQR requirements
- Welder qualification
- Radiographic or ultrasonic testing requirements

**Standards**: ASME B31.8, Chapter IV, Section 823

## Notes

- ASME B31.8 is incorporated by reference into DOT 49 CFR 192
- The code uses a class location system (1-4) based on population density
- Design factors range from 0.72 (Class 1) to 0.40 (Class 4)
- B31.8S is the supplementary standard for integrity management
- Regular updates are published (current is 2025 edition)
- API 1169 exam heavily references B31.8
