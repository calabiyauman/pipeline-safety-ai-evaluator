## Sample Test Questions (To Be Developed)

### Question 1: Smart Pig Selection
**Scenario**: A 30-year-old natural gas transmission pipeline requires integrity assessment:
- Diameter: 24 inches
- Wall thickness: 0.375 inches
- Coating: Coal tar enamel (old, deteriorated)
- Operating pressure: 800 psig
- Suspected issues: External corrosion, possible dents

**Task**: Recommend appropriate inline inspection tools and explain the selection rationale.

**Expected Elements**:
- MFL tool for metal loss detection
- Geometry tool for dent detection
- UT tool for crack detection (if suspected)
- Tool selection criteria per API 1163

**Standards**: API 1163, API 1160

---

### Question 2: ILI Data Analysis
**Scenario**: An MFL inspection report identifies the following anomalies:
- Feature A: 30% wall loss, 2 inches long, external
- Feature B: 50% wall loss, 4 inches long, internal
- Feature C: Dent, 5% of diameter depth, 12 inches long

**Task**: Prioritize repairs per API 1160 and determine response timelines.

**Expected Elements**:
- Severity assessment
- Remediation categories (immediate, scheduled, monitored)
- Response timeframes
- Re-inspection intervals

**Standards**: API 1160, ASME B31.8S

---

### Question 3: Pipeline Materials Selection
**Scenario**: A new sour gas pipeline is being designed:
- H2S content: 1,000 ppm
- CO2 content: 5%
- Operating pressure: 1,200 psig
- Temperature: 150°F

**Task**: Recommend appropriate pipe material and explain the selection criteria.

**Expected Elements**:
- Sour service requirements per NACE MR0175/ISO 15156
- Material hardness limitations (HRC ≤ 22)
- API 5L grade selection
- Corrosion-resistant alloy considerations

**Standards**: NACE MR0175/ISO 15156, API 5L

---

### Question 4: Hydrate Prevention
**Scenario**: An offshore gas pipeline operates at:
- Pressure: 1,500 psig
- Temperature: 40°F
- Gas composition: High methane with water vapor

**Task**: Determine the hydrate formation risk and recommend prevention measures.

**Expected Elements**:
- Hydrate formation conditions
- Methanol injection calculations
- Insulation requirements
- Operating envelope definition

**Standards**: Industry best practices, GPSA Engineering Data Book

## Notes

- Smart pigging is essential for modern pipeline integrity management
- Multiple ILI technologies exist (MFL, UT, EMAT, geometry)
- Tool selection depends on threat type, pipe characteristics, and operating conditions
- API 1163 provides qualification requirements for ILI tools
- API 1160 provides guidance on ILI data analysis and response
- Pipeline incidents often drive regulatory changes
- Industry associations (AGA, API, INGAA) publish best practice guidelines
