# API (American Petroleum Institute) Benchmark Test Sources

## Overview
API is the primary standards body for oil and natural gas pipeline operations. Their Individual Certification Programs (ICP) provide industry-recognized credentials for pipeline inspectors and professionals.

## Key Certifications & Resources Found

### 1. API 1169 - Pipeline Construction Inspector
- **URL**: https://www.api.org/products-and-services/individual-certification-programs/certifications/api1169
- **Description**: Certification for onshore new pipeline construction inspection
- **Exam Format**: 25 questions, 4 hours, open book
- **Body of Knowledge**:
  - API 1169 (Basic inspection requirements)
  - API 1110 (Pressure testing)
  - API Spec Q1 (Quality management)
  - API 1104 (Welding)
  - ASME B31.4/B31.8 (Pipeline codes)

### 2. API 1184 - Pipeline Facility Construction Inspector
- **URL**: https://www.api.org/products-and-services/individual-certification-programs/certifications/api1184
- **Description**: Inspection of pipeline facilities (stations, terminals)

### 3. API 1104 - Welding of Pipelines
- **URL**: https://www.api.org/products-and-services/standards/important-standards-announcements/1104
- **Description**: Primary welding standard for pipelines (22nd Edition available)
- **AWS CWI Exam**: 60 multiple-choice questions covering:
  - Materials and design
  - Fabrication
  - Inspection
  - Qualification
  - Reports and records

### 4. API Practice Tests
- **URL**: https://apiprep.com/api-practice-tests/
- **Description**: Third-party practice exam provider
- **Content**: Realistic practice questions for API certifications

### 5. API 1169 Sample Questions
- **URL**: https://inspector-training.com/api-1169-sample-questions/
- **Description**: Free sample questions for API 1169 exam

### 6. Atlas API Training
- **URL**: https://atlasapitraining.com/api-1169
- **Description**: Online training covering API 1169, API 1110, API Q1, API 1104, ASME B31.4

## Sample Test Questions (To Be Developed)

### Question 1: API 1104 Welding Requirements
**Scenario**: A pipeline project requires welding API 5L Grade X52 pipe with 0.375-inch wall thickness. The welding procedure specification (WPS) calls for E7018 electrodes.

**Task**: Determine the required preheat temperature and explain the qualification requirements for welders working on this project per API 1104.

**Expected Elements**:
- Preheat temperature based on material grade and wall thickness
- Welder qualification test requirements
- Essential variables that require requalification
- Acceptance criteria for welder qualification tests

**Standards**: API 1104, Section 6 (Welder Qualification)

---

### Question 2: API 1169 Inspection Planning
**Scenario**: You are assigned as the lead inspector for a 50-mile natural gas transmission pipeline construction project. The pipeline is API 5L Grade X65, 24-inch OD, 0.500-inch wall thickness, operating at 1,440 psig MAOP.

**Task**: Develop an inspection plan identifying critical inspection points, required hold points, and documentation requirements.

**Expected Elements**:
- Pre-construction activities (material verification, WPS review)
- During construction inspection points (trenching, welding, coating)
- Hold points (welder qualification, pressure test)
- Post-construction documentation

**Standards**: API 1169, API 1104, ASME B31.8

---

### Question 3: Pressure Testing per API 1110
**Scenario**: A new 16-inch crude oil pipeline requires hydrostatic testing. The pipeline specifications are:
- Material: API 5L Grade X60
- Wall thickness: 0.375 inches
- Design pressure: 1,000 psig
- Test section length: 10 miles
- Elevation change: +800 feet

**Task**: Calculate the required test pressure and develop a hydrostatic test procedure.

**Expected Elements**:
- Test pressure calculation per API 1110
- Elevation correction
- Hold time determination
- Acceptance criteria
- Safety procedures

**Standards**: API 1110, ASME B31.4
